== Description ==
Woocommerce esetén ha a vásárló megadja az irányítószámát, a plugin automatikusan beírja a hozzá tartozó várost a megfelelő helyre

== Installation ==
1. Csomagold ki a fájlokat a `woocommerce-hungarian-postcode` könyvtárban
1. Aktiváld a plugint a `Bővítmények` menüpont alatt1

== Frequently Asked Questions ==

= Lehet kézzel frissíteni a listát? =

Igen, a bővítmény mappájában lévő `postcode.json` fájlban írjuk át az irányítószámot vagy a várost.

= Adhatok hozzá egyedi irányítószámot / várost a listához? =

Igen, a a bővítmény mappájában lévő `postcode.json` fájlhoz fűzzük hozzá az egyedi tartalmakat, úgy, hogy figyeljünk a JSON fájl formai követelményeire. [JSON fájl](https://hu.wikipedia.org/wiki/JSON#Adatt%C3%ADpusok,_szintaxis_%C3%A9s_p%C3%A9lda "JSON - A Wikipédiából, a szabad enciklopédiából")
